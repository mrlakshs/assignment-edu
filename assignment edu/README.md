# Assignments

This repository contains production-grade solutions for two enterprise Data Science assignments. Both projects focus on scalable architectures, modular code, and real-world business constraints.

### Projects Included:

**1. [Predictive Maintenance: Server Failure Prediction](./question 1 server failure prediction)**
* **Objective:** Reduce server downtime by >40% across 50,000+ global servers.
* **Tech Stack:** SQL, LightGBM, Power BI.
* **Highlights:** Handles 25% missing telemetry data natively and achieves < 5-second inference times.

**2. [NLP: Intelligent IT Ticket Auto-Resolution](./question 2 it ticket resolution)**
* **Objective:** Automate the classification and resolution of 2 million monthly IT tickets.
* **Tech Stack:** Tesseract (OCR), Sentence-Transformers, FAISS (Vector DB).
* **Highlights:** Uses semantic search to handle noisy text and 5,000+ classes with sub-second latency.
